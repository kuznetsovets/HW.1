git config --global user.name - your global name git config --global user.email - your global email git init - Create an empty Git repository or reinitialize an existing git status - Show the working tree status git add - Add file contents to the index git commit - Record changes to the repository git log - Show commit logs git checkout - Switch branches or restore working tree files git branch - List, create, or delete branches git push - Update remote refs along with associated objects git pull - Fetch from and integrate with another repository or a local git merge - Join two or more development histories together git clone - Clone a repository into a new directory git ignore - Specifies intentionally untracked files to ignore git revert - Revert some existing commits git reset - Reset current HEAD to the specified state git rebase - Reapply commits on top of another base tip git cherry-pick - Apply the changes introduced by some existing commits git stash - Apply the changes introduced by some existing commits
